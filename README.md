### 起動
・Dockerをインストール  
・Docker Quickstart Terminalのアイコンをクリックして起動  
・Dockerコンソール上で
>docker run -it b.gcr.io/tensorflow/tensorflow:latest-devel  
  
を実行して最新版TensorFlowを導入
